# plate
A template management solution

## Installation
Run the install script
```bash
sudo ./install
```

## Usage
```bash
USAGE:
	plate [OPTIONS] TEMPLATE OUTPUTDIR

OPTIONS:
	-l	List available templates
	-p	Set project name
	-d	Set description text
	-V	Show version information
```
